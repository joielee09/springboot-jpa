### ✔ Installation

For development:

1. git clone https://github.com/joielee09/springboot-jpa.git </br>
2. cd springboot-jpa </br>
3. build gradle</br>
4. run</br>

For Production:

1. git clone https://github.com/joielee09/springboot-jpa.git </br>
2. cd springboot-jpa </br>
3. build gradle</br>
4. ./gradlew.bat build(For Winow) </br>
5. cd build/libs</br>
6. java -jar hello-spring-0.0.1-SNAPSHOT.jar</br>
